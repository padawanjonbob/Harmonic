# 💧 Harmonic Weight Loss Tracker

A simple desktop app for meal tracking using the Harmonic Weight Loss Formula:

## Core Formula

Liquid ÷ Food ≥ 2.5

Example:

- 100g food = 250mL liquid  
- 400g food = 1000mL liquid

Use transparent water-based liquids such as:

- Water
- Broth
- Clear soup
- Tea
- Coffee (light use)
- Zero calorie beverages

# How To Use

1. Enter food amount in grams
2. Enter liquid amount in mL
3. Optionally enter body weight
4. Click **Calculate**
5. Click **Save Meal**

Track your daily ratio goal:

  Liquid / Food ≥ 2.5

# Example

Meal:

* Food = 500g
* Liquid = 1250mL

Ratio:

1250 / 500 = 2.5 ✅
